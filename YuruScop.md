- EdgeDB 백엔드
  - SQLite
    - 각 테이블의 메타데이터 저장
      - 마이그레이션
    - SQL 실행
    - 결과 정보 넘겨주기
  - EdgeQL(DDL 포함)
    - 렉서 및 파서
    - (SQLite 호출) 현재 DB 스키마 가져오기
    - 중간 트리 제작
    - SQL 제작
    - (SQLite 호출) 적용
- EdgeDB 프론트엔드
  - CLI
    - Shell
      - REPL
      - 마이그레이션
        - migration apply
          - (SQLite에서 메타데이터 호출) 적용안 된 마이그레이션 파일 찾기
          - EdgeQL 호출
      - 쿼리
        - 쿼리 파일 경로 입력 받기
        - 쿼리문 입력 받기
        - 아웃풋 형태 입력 받기
        - (EdgeQL 호출) 적용
      - 인포
        - 현재 서비스 정보
      - 인스턴스
        - 현재 파일(`*.sqlite`)
